# Data and Replication Code for Practicing with LLMs Cultivates Human Empathic Communication

## Repository Contents

- `analysis.ipynb`: Main replication notebook with all analyses and figures.
- `data/experiment_1.csv`: Conversation-level data from the Lend an ear experiment
- `data/experiment_2.csv`: Human preference dataset.
- `data/elo_rankings.csv`: Elo rankings for conversations.
- `data/all_themes.csv`: Thematic coding of supporter responses in Lend an ear.
